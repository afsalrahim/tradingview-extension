# TradingView Watchlist Chrome Extension

A tradingview extension to automate the creation of watchlists from screener results.

## Features

    - Create watchlists from screener results
    - Add sections to watchlists
    - Rename sections
    - Add stocks to watchlists

## Installation

    - Download the extension from the Chrome Web Store
    - Open Chrome and navigate to chrome://extensions/
    - Enable Developer Mode
    - Click 'Load unpacked' and select the extension directory
    - The extension should now be installed and ready to use

## Usage

    - Open TradingView and navigate to the screener
    - Click the 'Watchlists' button
    - Click the 'Create new list' menu item
    - Add stocks to watchlists
